# COUNTRY BALL WARS - 2D STRATEJİ OYUNU

## Proje Hakkında
Bu proje, Unity ve C# kullanılarak geliştirilmekte olan 2D strateji tabanlı bir savaş oyunudur.

Oyunda farklı karakterler arasında savaş mekanikleri, hasar sistemi ve temel strateji unsurları üzerinde çalışılmaktadır.

Proje şu anda geliştirme aşamasındadır ve aktif olarak yeni özellikler eklenmektedir.

---

## Kullanılan Teknolojiler
- Unity
- C#
- Object-Oriented Programming (OOP)
- Unity Physics & Collision System
- Unity UI System

---

## Geliştirilen Özellikler
- Temel saldırı sistemi
- Hasar hesaplama mekanikleri
- Collision (çarpışma) sistemi
- Event tabanlı savaş mekanikleri
- Scene yönetimi
- GameObject yönetimi
- Temel kullanıcı arayüzü (UI)
- OOP prensiplerine uygun oyun mimarisi

---

## Devam Eden Geliştirmeler
- Gelişmiş yapay zekâ sistemi
- Yeni karakterler ve yetenekler
- Daha gelişmiş kullanıcı arayüzü
- Oyun dengesi düzenlemeleri
- Ses efektleri ve animasyon geliştirmeleri
